# Contracts

| Contract | Explanation |
|--------------------------|----------------------------------------------|
| Invoker                  | Main entrypoint for Kollateral, handles aggregation and flash loan invocation. |
| AaveLiquidityProxy       | Proxy for sourcing liquidity from Aave. |
| SoloLiquidityProxy       | Proxy for sourcing liquidity from Solo (DyDx). |
| KollateralLiquidityProxy | Proxy for sourcing liquidity from Kollateral's kTokens. |
| kEther                   | Simple Ether collateral pool with flash loans for testing. |
| kUSDC                    | Simple USDC collateral pool with flash loans for testing. |
| kDAI                     | Simple DAI collateral pool with flash loans for testing. |
